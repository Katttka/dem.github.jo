🕸️ Как работает нейронная сеть? (Простое объяснение)

В отличие от линейной регрессии (одна формула) и дерева решений (правила «если-то»), нейросеть учится находить сложные нелинейные закономерности через слои простых вычислений.

Вход
возраст
стаж
math
...
→
Скрытый слой
нейрон ①
нейрон ②
нейрон ③
нейрон ④
→
Выход
прогноз: 78.4
Что происходит:
1. Каждый вход × вес (важность признака)
2. Сумма → функция активации (ReLU: max(0, x))
3. Нейроны комбинируют сигналы → находят паттерны
4. Выходной слой выдаёт финальный прогноз
📐 Линейная регрессия
y = w₁x₁ + w₂x₂ + ... + b

• Одна формула для всех данных
• Быстрая, но не ловит сложные зависимости
🌳 Дерево решений
если возраст > 25 → если стаж < 3 → ...

• Набор правил «если-то»
• Интерпретируемо, но может переобучаться
🕸️ Нейросеть
слой₁ → активация → слой₂ → ... → выход

• Многослойная композиция функций
• Универсальный аппроксиматор, «чёрный ящик» 
import torch.nn as nn

class StudentNet(nn.Module):
    def __init__(self, input_features):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(input_features, 8),   # вход → 8 нейронов
            nn.ReLU(),                            # активация
            nn.Linear(8, 4),                      # скрытый слой
            nn.ReLU(),
            nn.Linear(4, 1)                       # выход: прогноз
        )
    def forward(self, x):
        return self.net(x)
